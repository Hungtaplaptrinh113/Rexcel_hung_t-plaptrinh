# Ứng dụng Streamlit đọc file Excel đơn giản

## Cài đặt

Trong thư mục dự án, mở terminal và chạy:

```bash
pip install -r requirements.txt
```

## Chạy ứng dụng

Trong cùng thư mục, chạy:

```bash
streamlit run app.py
```

Sau đó mở trình duyệt theo đường link Streamlit hiển thị (thường là `http://localhost:8501`).
